# Core 子系统

## 职责

Core 子系统对应 src/LightyDesign.Core。它用于承载 LightyDesign 的核心领域模型、文件协议、表头语义、数据值解析和共享基础类型。

## 当前已完成的工作

1. 已创建 .NET 类库项目骨架。
2. 已加入解决方案，并作为其他后端项目的基础引用。
3. 已完成与 Generator、DesktopHost、Tests 的项目依赖连接。
4. 已验证解决方案可成功构建。

## 当前尚未实现的业务能力

1. 工作区模型与工作簿模型。
2. 表头定义与表头反序列化模型。
3. txt 数据行解析与转义/反转义逻辑。
4. 引用语法 [[...]] 的解析模型。
5. 面向 Generator 和 DesktopHost 的共享接口。

## 当前状态结论

Core 目前已经完成工程层面的落位和依赖关系接入，但业务模型仍处于待实现状态。后续所有文件协议和数据规则都应该优先在这里沉淀，而不是直接写进 UI 或宿主层。

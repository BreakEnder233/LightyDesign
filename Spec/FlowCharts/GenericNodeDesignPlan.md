# 标准节点参数化设计计划

## 背景

当前节点定义一旦把 `List<int32>`、`List<float>`、`Dictionary<string, int32>` 等具体类型直接写死到端口上，就会导致同一类节点为了不同元素类型反复复制。`Add`、`GetAt`、`Set`、`ForEach` 这类标准节点最容易出现这类重复。

本轮设计的目标不是把所有端口都降成 `object`，而是让“节点可复用”和“连线可分析”同时成立。

## 方案结论

1. 不建议把 List/Dictionary 操作节点统一降成 `object -> object`，再把类型错误完全推迟到 C# 编译期。
2. 更合适的方向是“参数化类型 + 局部类型推断 + 标准节点重载族”。
3. 容器操作节点应优先使用参数化类型复用，例如 `List<T>`、`Dictionary<TKey, TValue>`。
4. 算术、比较、相等等标准计算节点应采用“同一节点定义 + 多个受控签名”的重载族模型，而不是为每个具体类型复制一份节点定义。
5. 生成器应在导出前先把节点实例解析成唯一的具体签名；目标语言里的重载函数只是代码落地手段，不应成为第一道类型校验器。

## 核心设计

### 1. 节点定义支持类型参数

节点定义新增 `typeParameters`，并允许 `TypeRef.kind = typeParameter`。这使一个节点定义可表达：

1. `List.Add<T>`
2. `List.GetAt<T>`
3. `Dictionary.Set<TKey, TValue>`
4. `Dictionary.TryGetValue<TKey, TValue>`

### 2. 节点实例允许保留显式类型提示

流程图实例中的节点可选持久化 `typeArguments`。如果节点尚未通过连线完全推断出具体类型，编辑器仍可允许用户手动指定 `T=int32`、`T=float` 等提示。

### 3. 校验拆成两层

1. 结构校验：节点/端口存在性、方向、基数、计算回路等，继续作为保存阻断错误。
2. 语义校验：类型参数求解、约束检查、重载候选收敛与二义性诊断。

其中：

1. 明确冲突应阻止保存。
2. 仅“尚未定型”但未冲突的节点可先保存为带诊断状态的草稿。
3. 导出代码前必须把所有节点实例解析成唯一具体签名，否则阻止导出。

### 4. 标准节点通过稳定绑定名接入生成器

节点定义不直接把目标语言函数名写死，而是使用稳定的 `codegenBinding` 标识，例如：

1. `List.Add`
2. `Dictionary.Set`
3. `Arithmetic.Add`
4. `Compare.Equal`

生成器根据绑定名与已解析出的具体类型，决定输出：

1. 泛型 helper 调用
2. 具体重载 helper 调用
3. 或生成到节点 partial class 的固定模板代码

## 分阶段实施

### 阶段 1：协议收敛

1. 在 `TypeSystem` 中引入 `typeParameter`。
2. 在节点定义中引入 `typeParameters` 和 `codegenBinding`。
3. 在流程图实例中引入可选 `typeArguments`。
4. 明确结构校验与语义校验的分层规则。

### 阶段 2：Core 与编辑器校验器

1. 为单个节点实例建立类型参数替换表。
2. 从属性值、显式 `typeArguments`、上下游连线联合推断具体类型。
3. 对标准重载族做候选签名筛选。
4. 把“冲突”“未定型”“二义性”区分成不同诊断等级。

### 阶段 3：标准节点库收敛

1. 把 `List` / `Dictionary` 基础节点收敛为参数化标准节点。
2. 把 `Add`、比较、相等等计算节点收敛为少量重载族。
3. 冻结一份标准节点绑定名清单，避免生成器靠节点中文名或路径猜语义。

### 阶段 4：生成链路落地

1. 导出前先完成节点实例具体化。
2. 容器节点优先生成为泛型调用。
3. 数值运算节点优先生成为显式重载调用，不依赖运行时再做二次分派。
4. 未解析或二义性的节点实例直接阻止导出。

### 阶段 5：测试补齐

1. 节点定义协议读取测试。
2. 节点实例类型推断测试。
3. 重载族候选收敛测试。
4. 保存期诊断与导出期阻断测试。
5. 生成代码中具体 helper / overload 选路测试。

## 当前决定

本次规范调整先明确方向：

1. 不采用“默认全部 object 化”的弱类型方案。
2. 采用“类型参数 + 局部推断 + 重载族绑定”的强约束方案。
3. 保留保存期结构合法性分析，并补充保存期语义诊断与导出期最终收敛。
